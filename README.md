# Playwright + TypeScript QA Automation Framework

Enterprise-style QA Automation Framework built with Playwright + TypeScript following scalable automation best practices.

## Features

- UI Automation Testing
- API Testing
- Cross-browser execution
- GitHub Actions CI/CD
- Playwright HTML Reports
- Page Object Model (POM)
- Reusable Fixtures
- Scalable framework architecture
- Smoke & Regression ready structure

---

## Tech Stack

- Playwright
- TypeScript
- Node.js
- GitHub Actions

---

## Project Structure

```bash
.
├── pages
├── tests
│   ├── api
│   └── ui
├── test-data
├── playwright.config.ts
├── package.json
└── .github/workflows
```

---

## Installation

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```

## Run Headed Mode

```bash
npx playwright test --headed
```

## Open HTML Report

```bash
npx playwright show-report
```

---

## CI/CD

This framework includes GitHub Actions integration for automated test execution on push and pull requests.
